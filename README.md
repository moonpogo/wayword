# Wayword

Write. See what emerges.

Wayword is a minimal writing environment that reflects your language back to you in real time.

It is not a notes app.
It is not a document editor.

It is a space for observation.

---

## What it does

- Surfaces patterns in your writing as you type
- Highlights repetition, structure, and variation
- Tracks how language shifts across a session
- Returns simple, directional feedback after each run

The goal is not to optimize writing.

The goal is to make you aware of how you write.

---

## Core loop

1. Receive a prompt
2. Write without interruption
3. Submit
4. See what emerges
5. Press Enter to begin again

---

## Design principles

- Minimal interface, no clutter
- Feedback over instruction
- Observation over judgment
- Behavior over theory

Everything in the app exists to support the writing loop.

---

## Features

- Real-time text analysis
- Highlighting for repetition and structure
- Session feedback (score, variety, sentence shape)
- Prompt system across multiple modes
- Keyboard-driven interaction (Enter to restart)
- Lightweight profile view of writing patterns

---

## Philosophy

Wayword treats writing as a system you can observe.

Not something to perfect.
Something to notice.

Patterns reveal themselves through repetition.
Clarity comes from seeing them.

---

## Status

Active development.

This is an evolving tool. Features are being added carefully to preserve the core experience.

---

## Tech

- HTML / CSS / JavaScript
- No frameworks
- Designed for speed and simplicity

---

## Usage

Open the app and begin writing.

Press Enter to submit a run.
Press Enter again to begin a new one.
